# SISTEMA-LOGIN-PYTON-1.0
Projeto simples em Python simulando um sistema de login com cadastro de usuários via terminal.
 Sistema de Login em Python

Meu primeiro projeto em Python desenvolvido durante os estudos de Análise e Desenvolvimento de Sistemas.

## Funcionalidades

* Cadastro de usuários
* Login de usuários
* Menu interativo no terminal
* Validação de usuário e senha

## Tecnologias

* Python 3

## Objetivo

Praticar lógica de programação utilizando:

* Variáveis
* Listas
* Estruturas condicionais (`if`, `elif` e `else`)
* Estruturas de repetição (`while`)
* Entrada e saída de dados (`input` e `print`)
